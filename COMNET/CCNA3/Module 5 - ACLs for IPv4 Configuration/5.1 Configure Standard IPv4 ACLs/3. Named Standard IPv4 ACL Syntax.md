To create a Named Standard ACL, use the `ip access-list standard` command.

ACL names are:
- Alphanumeric
- Case Sensitive
- Must be Unique

Capitalizing ACL names is Not Required but makes them <mark class="hltr-green">stand out</mark> when **viewing** the **running-config** output.
